# Smart Static Website with AWS

This is my first cloud project showcasing a secure, fast static website hosted using:

- **Amazon S3** (Static website hosting)
- **CloudFront** (CDN with HTTPS)
- **AWS WAF** (Security)
- **CloudWatch** (Monitoring)
